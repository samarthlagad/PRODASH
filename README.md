✦ ProDash
Productivity Reimagined. A premium, single-file productivity powerhouse featuring Kanban boards, Markdown notes, interactive calendars, and real-time analytics.

ProDash is a "Luxury Obsidian" inspired productivity suite that requires zero installation, zero dependencies, and zero configuration. It operates entirely in your browser using local persistence, keeping your data private and your workflow lightning-fast.

✨ Key Features
🧱 Dynamic Kanban Engine: Organize tasks across 'Todo', 'Doing', and 'Done' columns with a fluid drag-and-drop interface.

📝 Markdown-Powered Notes: A split-pane editor for writing in Markdown with real-time preview and instant local saving.

📅 Interactive Calendar: Manage events and deadlines with a visual monthly overview and quick-add functionality.

📈 Real-time Analytics: Visualize your productivity with automated charts and performance statistics.

⌨️ Command Palette (⌘K): Navigate the entire app and trigger actions instantly via a global search and command interface.

🎨 Obsidian Luxury Design: A stunning glassmorphic UI with animated mesh backgrounds, grain overlays, and full Dark/Light mode support.

🔒 Local Persistence: Powered by IndexedDB, ensuring all your data stays on your machine and survives browser refreshes.

🛠️ Technology Stack
Built with a focus on "Extreme Portability" and "Pure Performance":

Frontend: Vanilla JavaScript (ES6+), HTML5, CSS3.

Database: IndexedDB (Browser-native local storage).

Typography: Syne (Display), DM Sans (Body), DM Mono (Interface).

Icons: Custom SVG-based iconography.

🚀 Getting Started
Since ProDash is a single-file application, there is no build process:

Download the prodash-2.html file.

Open the file in any modern web browser (Chrome, Firefox, Safari, or Edge).

Start organizing. Your data is automatically saved to your browser's internal database.

⌨️ Shortcuts
Efficiency is at the heart of ProDash:

⌘K or Ctrl+K: Open Command Palette.

/: Focus Global Search.

Esc: Close Modals/Palettes.
